Git repository

Powershellen belül:

  1. Készítsünk egy mappát, ami tartalmazza a projektünket
  2. Lépjünk be a mappába
  3. Írjuk be: git init
  4. Írjunk a projektbe valamilyen kódot
  5. Írjuk be: git add, hogy hozzáadja a fájlokat
  6. Írjuk be: git commit

Githubon:

  1. Menjünk a profilunkra
  2. Kattintsunk a Respository fülre
  3. Hozzunk létre egy respositoryt
  4. Végezzük el a munkát
  5. Commitoljuk a változtatásokat
  6. Pusholjuk ki őket
